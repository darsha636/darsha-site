# Darsha Store

## Project Overview

Darsha Store is an e-commerce web application developed during my internship.

### Technologies Used

- HTML
- CSS
- JavaScript
- Java (HttpServer)

## Features

- Product Listing
- Product Details
- Search Products
- Shopping Cart
- User Login
- Loan Eligibility Feature

## Deployment

The live version of this project is deployed using Vercel.

Since Vercel does not support hosting a Java HttpServer backend, the deployed version uses a frontend-only implementation with JavaScript.

## Original Java Backend

The original Java backend developed during my internship has been preserved in the `backend-java` folder.

Files included:

- Main.java
- WebServer.java
